# Bike Sales Analysis (Excel project)
## objective
Analyze bike sales and customer data to identify:
- purchase patterns 
- customer segmentation
- Business insights
 ## TOOLS USED:
- MICROSOFT EXCEL
- Pivot Tables
- Charts
- Filters
## Data set
- Customer demographics
- Income
- Region
- Age
  ## Analysis performed
  - Income vs purchase behavior analysis
  - Age group segmentation
  - Region-wise sales distribution
  - Customer segmentation using pivot tables
   ## Key insights
  - Customers with high income show higher bike purchase probability
    ## Files
  - bike_sales_raw_data.xlsx  "raw data set"
  - bike_sales_analysis.xlsx  "cleaned data + pivot analysis"
  - bike_sales_dashboard.xlsx  "dashboard file"
